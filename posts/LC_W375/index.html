Q1. 🟢 2960. Count Tested Devices After Test Operations

題意

給定一個長度為 nnn 的整數陣列 batteryPercentagesbatteryPercentagesbatteryPercentages ，表示 nnn 個裝置的電池百分比，你的任務是按照順序測試每個設備 iii，執行以下測試操作：

  • 如果 batteryPercentages[i]>0batteryPercentages[i] > 0batteryPercentages[i]>0：
    • 增加答案數，表示已測試設備的計數。
    • 將下標在 [i+1,n−1][i + 1, n - 1][i+1,n−1] 的所有設備的電池百分比減少 111，確保它們的電池百分比不會低於 000 ，即 batteryPercentages[j]=max(0,batteryPercentages[j]−1)batteryPercentages[j] = max(0, batteryPercentages[j] - 1)batteryPercentages[j]=max(0,batteryPercentages[j]−1)。
    • 移動到下一個裝置。
  • 否則，請移動到下一個設備而不執行任何測試。
    返回已測試設備的數量。

思路：直接模擬 / 差分陣列

方法一：直接模擬

  • 按照題目要求模擬即可
  • 時間複雜度 O(n2)O(n^2)O(n2)

方法二：差分陣列

  • 對於每個設備，其需要減少的電量為前面已測試設備的數量，因此若前面已測試設備的數量大於當前電量，則當前設備不需要測試。
  • 時間複雜度 O(n)O(n)O(n)

Q3. 🟡 2962. Count Subarrays Where Max Element Appears at Least K Times

題意

給定一個整數陣列 numsnumsnums 和一個正整數 kkk。統計有多少個子陣列滿足「 numsnumsnums 中的最大元素至少出現 kkk 次」，並返回滿足這一條件的子陣列的數目。

思路：滑動窗口 / 前綴和 + 二分搜索

方法一：滑動窗口

  • 讓窗口保持在向左擴張1次就能滿足條件的狀態，即保證 [left−1,right][left-1, right][left−1,right] 中 mxmxmx 出現的次數恰好為 kkk 。
  • 由於上述性質，對於每個 rightrightright ，都可以往左擴張 leftleftleft 次，使其滿足題目條件，因此答案增加 leftleftleft。
  • 以 nums=[1,3,2,3,2,3]nums=[1,3,2,3,2,3]nums=[1,3,2,3,2,3], k=2k=2k=2 為例，mx=3mx=3mx=3：
    • right=0right=0right=0，left=0left=0left=0，cnt=0cnt=0cnt=0，left=0left=0left=0，ans+=0=0ans+=0=0ans+=0=0。
    • right=1right=1right=1，left=0left=0left=0，cnt=1cnt=1cnt=1，left=0left=0left=0，ans+=0=0ans+=0=0ans+=0=0。
    • right=2right=2right=2，left=0left=0left=0，cnt=1cnt=1cnt=1，left=0left=0left=0，ans+=0=0ans+=0=0ans+=0=0。
    • right=3right=3right=3，left=2left=2left=2，cnt=1cnt=1cnt=1，left=2left=2left=2，ans+=2=2ans+=2=2ans+=2=2。
    • right=4right=4right=4，left=2left=2left=2，cnt=1cnt=1cnt=1，left=2left=2left=2，ans+=2=4ans+=2=4ans+=2=4。
    • right=5right=5right=5，left=4left=4left=4，cnt=1cnt=1cnt=1，left=4left=4left=4，ans+=4=8ans+=4=8ans+=4=8。
  • 時間複雜度 O(n)O(n)O(n)

方法二：前綴和 + 二分搜索

  • 先用前綴和 preprepre 統計 numsnumsnums 中最大元素出現的次數，則 pre[i]pre[i]pre[i] 表示 nums[:i]nums[:i]nums[:i] 中 mxmxmx 出現的次數。
  • 對於每個左端點 iii，找到 pre[i]+kpre[i] + kpre[i]+k 的位置 jjj，則 nums[i:j]nums[i:j]nums[i:j] 中 mxmxmx 出現的次數恰好為 kkk，subarray可以繼續向右，故延伸答案增加 n−(j−1)n - (j - 1)n−(j−1)。
  • 還是以 nums=[1,3,2,3,2,3]nums=[1,3,2,3,2,3]nums=[1,3,2,3,2,3], k=2k=2k=2 為例，mx=3mx=3mx=3，pre=[0,0,1,1,2,2,3]pre=[0, 0, 1, 1, 2, 2, 3]pre=[0,0,1,1,2,2,3]：

    • i=1i=1i=1，j=4j=4j=4，nums[i:j]=[3,2,3]nums[i:j]=[3,2,3]nums[i:j]=[3,2,3]，ans+=3=6ans+=3=6ans+=3=6。
    • i=2i=2i=2，j=6j=6j=6，nums[i:j]=[2,3,2,3]nums[i:j]=[2,3,2,3]nums[i:j]=[2,3,2,3]，ans+=1=7ans+=1=7ans+=1=7。
    • i=3i=3i=3，j=6j=6j=6，nums[i:j]=[3,2,3]nums[i:j]=[3,2,3]nums[i:j]=[3,2,3]，ans+=1=8ans+=1=8ans+=1=8。
    • i=4i=4i=4，j=7j=7j=7，nums[i:j]=[2,3]nums[i:j]=[2,3]nums[i:j]=[2,3]，ans+=0=8ans+=0=8ans+=0=8。(此時已經找不到滿足條件的j了)
